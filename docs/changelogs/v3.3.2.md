# Changelog v3.3.2 (March 14, 2026)

LedgerMind v3.3.2 is a maintenance release that automates Gemini CLI hook configuration and refines default settings for better compatibility.

## 🚀 Highlights

### ⚡ Installer Automation

- **Zero-Touch Hook Configuration** — The installer now automatically updates `settings.json` for Gemini CLI, configuring both `BeforeAgent` and `AfterAgent` hooks during installation.
- **Relative Path Support** — Project-specific installations now use relative paths for hook scripts, ensuring hooks work correctly when the project is moved or accessed from different environments.
- **Robust Config Parsing** — Improved JSON parsing for existing settings files, preventing data loss when adding hooks to existing configurations.

### 🛠️ Configuration Refinement

- **Flexible Model Selection** — Removed hardcoded `gemini-2.0-flash` default from `GeminiConfigManager` to allow system-level or environment-specific defaults to take precedence when no explicit configuration is provided.
- **Simplified MCP Config** — Removed redundant `disabled` flag from default MCP server registration.

## 🔧 Technical Details

### Automatic Hook Registration

The `GeminiInstaller` now handles the registration of LedgerMind as a hook provider for Gemini CLI:

```python
# Automatic registration of hooks in settings.json
config["hooks"]["BeforeAgent"] = [
    {
        "matcher": "*",
        "hooks": [
            {
                "name": "ledgermind-context",
                "type": "command",
                "command": f"python3 {hook_rel_path} before"
            }
        ]
    }
]
```

## 📝 Migration from v3.3.1

**No migration required!** This release is fully backward compatible.

To automatically configure hooks for an existing project:
```bash
python3 -m ledgermind.server.installers install --mode project
```
