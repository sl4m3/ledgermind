# Changelog v3.3.5 (March 17, 2026)

LedgerMind v3.3.5 introduces Google AI Studio integration, enhanced Claude CLI support, and a robust API key management system.

## 🚀 Highlights

### 🧪 Google AI Studio Integration (Project Prism)

- **Native Support** — Added `AIStudioClient` for direct integration with Google AI Studio API. This provides a high-performance alternative to CLI-based enrichment.
- **Auto-Validation** — The `ledgermind init` command now includes a model validation step that verifies AI Studio model availability and responsiveness before saving the configuration.
- **Flexible Provider Selection** — Users can now choose between `cli`, `openrouter`, and `aistudio` during project initialization.

### 🛡️ Enhanced Claude CLI Support

- **Smart CLI Flags** — `CloudLLMClient` now automatically detects the active client and applies optimized flags. For Claude, it uses non-interactive mode (`-p`), disables tools, and enforces a strict empty MCP configuration to prevent infinite recursion during enrichment.
- **Client-Specific Model Routing** — Improved model selection logic in `EnrichmentConfig`. Each client (Claude, Gemini, Cursor) can now have its own dedicated enrichment model configured via `enrichment_model_{client}`.

### 🔒 Robust API Key Management

- **Multi-Source Discovery** — New `api_keys.py` utility allows LedgerMind to find API keys (like `OPENROUTER_API_KEY` or `GOOGLE_API_KEY`) in environment variables OR shell configuration files (`.bashrc`, `.zshrc`, etc.).
- **Interactive Init Fixes** — The initialization process now proactively helps users find or set their API keys, including a "source your config" tip.

## 🔧 Technical Details

### AI Studio Provider Configuration

The new provider can be configured via `ledgermind settings`:

```bash
ledgermind settings set enrichment_provider aistudio
ledgermind settings set aistudio_model gemini-1.5-pro
```

### Claude CLI Safety Mechanism

To avoid infinite loops where Claude-in-CLI calls LedgerMind-in-MCP, the enrichment process now uses these flags for Claude:
`--tools "" --strict-mcp-config --mcp-config '{"mcpServers":{}}'`

## 📝 Migration from v3.3.4

**Backward compatible.** New settings will use default values. To switch to AI Studio, run:
```bash
ledgermind init
```
