# Changelog v3.2.1 (March 2, 2026)

LedgerMind v3.2.1 introduces advanced knowledge processing capabilities through
iterative enrichment and architectural improvements to asynchronous LLM
operations.

## Highlights

### 🧠 Iterative Knowledge Enrichment
- **Recursive Refinement:** Implemented a recursive process for refining
  knowledge artifacts, allowing the system to build upon existing insights for
  deeper understanding.
- **Compressive Rationale:** Introduced a high-signal compression model for
  rationales, condensing complex technical logs into concise, actionable
  intelligence without losing critical context.

### ⚡ Reliable Asynchronous Operations
- **Non-Blocking Enrichment:** Re-engineered the LLM enrichment pipeline to
  operate reliably in the background across all knowledge types.
- **Enhanced Error Resilience:** Improved stability for asynchronous calls,
  ensuring that background processing doesn't interfere with core system
  performance or user interactions.

## What's Changed

- feat(core): implement iterative knowledge enrichment and compressive rationale
  by @sl4m3
- fix(core): enable reliable and asynchronous LLM enrichment for all knowledge
  types by @sl4m3
