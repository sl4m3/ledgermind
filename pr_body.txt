## 🎨 Palette: Enable Tab Completion for Path Prompt

**💡 What**: Changed `questionary.text()` to `questionary.path()` for the URL/path prompt in the `cli.py` configuration wizard.

**🎯 Why**: When users enter a local `.gguf` absolute path, they can now use tab-completion, which significantly reduces friction and errors when typing long paths.

**📸 Before/After**: N/A (CLI interaction behavior change)

**♻️ Accessibility**: Improves usability for all users, reducing keystrokes and typos.
